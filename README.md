# AYRES

A take on JARVIS independently developed by me.

**Currently it is still under development and nowhere near usable**

Right now I am still working on the interal memory and event managment system, and have not incorporated any AI just yet



# What can AYRES do?

Right now? Nothing.
However, in the future I will give it:
- Long-term and short-term memory stores, and the ability to access either
- Event and alarm managment, as to keep track of context of important events in the user's life (this was inspired by the scene in **Iron Man**, where JARVIS asks Tony how his event went)
- A hybrid system where the memory, events and other low-processing tasks are handled by a local model, and the communication and chatting will be handled by a cloud model (Might change this later if I get a better PC)
So far, thats about all my ideas, but I'll keep this repo posted
